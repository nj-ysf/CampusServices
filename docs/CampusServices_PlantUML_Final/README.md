# CampusServices — PlantUML diagrams

Final clean UML/conception diagrams for the CampusServices Java OOP + JavaFX project.

## Files

1. `01_use_case.puml` — Use case diagram
2. `02_class_domain.puml` — Domain class diagram
3. `03_class_architecture.puml` — JavaFX MVC + DAO architecture class diagram
4. `04_package_diagram.puml` — Package diagram
5. `05_sequence_auth_etudiant.puml` — Student authentication sequence
6. `06_sequence_emprunt_ressource.puml` — Resource loan sequence
7. `07_sequence_reservation_salle.puml` — Room reservation sequence
8. `08_sequence_demande_administrative.puml` — Administrative request submission sequence
9. `09_sequence_traitement_backoffice_demande.puml` — Back-office request processing sequence
10. `10_state_reservation_salle.puml` — Room reservation state diagram
11. `11_state_demande_administrative.puml` — Administrative request state diagram
12. `12_state_emprunt.puml` — Loan state diagram
13. `13_activity_parcours_etudiant.puml` — Student activity workflow
14. `14_activity_backoffice.puml` — Back-office activity workflow
15. `15_erd_mcd.puml` — ERD / MCD
16. `16_deployment.puml` — Deployment diagram

## Suggested repo location

Put this folder in:

```text
docs/uml/
```

## Rendering

With PlantUML installed:

```bash
plantuml docs/uml/*.puml
```

Or render online using PlantText, Kroki, or the VS Code PlantUML extension.
